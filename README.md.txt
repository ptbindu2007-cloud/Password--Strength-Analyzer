# Password Strength Analyzer

## Description
A Python-based tool designed to evaluate the strength of a password based on specific security criteria such as length, character diversity, and complexity.

## Features
* Checks password length.
* Detects the presence of uppercase and lowercase letters.
* Identifies numeric digits and special characters.
* Provides a strength rating (e.g., Weak, Moderate, Strong).

## How to Run
1. Ensure you have Python installed on your system.
2. Clone this repository: `git clone [your-repository-link]`
3. Run the script: `python password_analyzer.py`

## Requirements
* Python 3.x
